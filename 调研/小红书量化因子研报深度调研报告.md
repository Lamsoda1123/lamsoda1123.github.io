### 2.3 因子详解（部分代表性因子）

#### Factor 1: alpha_001（开盘缺口因子）

**中文名称**：开盘缺口反转因子

**计算逻辑**：
```
alpha_001 = rank(ts_argmax(power(((returns < 0) ? (ts_stddev(returns, 20)) : close), 2), 5)) - 0.5
```

**中文解释**：
1. 首先计算20日收益率标准差
2. 如果当日收益率为负，则使用该值；否则使用收盘价
3. 对结果平方
4. 取过去5日的最大值位置
5. 横截面排名并减0.5

**经济意义**：
- 捕捉开盘缺口反转效应
- 反映短期过度反应后的价格修正

**IC值**：0.023（根据笔记中用户透露的回测数据）

#### Factor 2: alpha_002（成交量加权收益率）

**中文名称**：成交量加权收益率因子

**计算逻辑**：
```
alpha_002 = -1 * correlation(rank(delta(log(volume), 2)), rank(((close - open) / open)), 6)
```

**中文解释**：
1. 计算成交量的2阶差分
2. 计算开盘到收盘的收益率
3. 对两者分别进行横截面排名
4. 计算6日相关系数
5. 取负值

**经济意义**：
- 成交量变化与收益率的负相关关系
- 反映"量在价前"的经典理论

**回测效果**：
- 年化收益率：8.3%

- 最大回撤：-4.2%

#### Factor 3: alpha_003（价格动量）

**中文名称**：价格动量因子

**计算逻辑**：
```

```

**中文解释**：
1. 对开盘价和成交量分别进行横截面排名
2. 计算10日相关系数
3. 取负值

**经济意义**：

### 4.3 因子挖掘方法论

#### Step 1: 基础指标池构建

**价格类指标**：
- 开盘价 open
- 最高价 high
- 最低价 low
- 收盘价 close
- 均价 vwap

**成交量类指标**：
- 成交量 volume
- 成交金额 amount
- 换手率 turnover

**时间类指标**：
- 时间戳 timestamp
- 交易日 weekday
- 月 month

#### Step 2: 算子库构建

**一元算子**：
- `delta(x, n)` = x - x.shift(n)
- `pct_change(x, n)` = (x - x.shift(n)) / x.shift(n)
- `sign(x)` = 1 if x > 0 else -1 if x < 0 else 0
- `abs(x)` = |x|
- `log(x)` = ln(x)
- `rank(x)` = x在横截面的分位数排名

**二元算子**：
- `multiply(x, y)` = x * y
- `divide(x, y)` = x / y
- `add(x, y)` = x + y
- `subtract(x, y)` = x - y
- `greater(x, y)` = 1 if x > y else 0

**时序算子**：
- `ts_delay(x, n)` = x.shift(n)
- `ts_delta(x, n)` = x - x.shift(n)
- `ts_mean(x, n)` = x.rolling(n).mean()
- `ts_stddev(x, n)` = x.rolling(n).std()
- `ts_corr(x, y, n)` = x.rolling(n).corr(y)
- `ts_covar(x, y, n)` = x.rolling(n).cov(y)
- `ts_rank(x, n)` = x.rolling(n).rank()
- `ts_max(x, n)` = x.rolling(n).max()
- `ts_min(x, n)` = x.rolling(n).min()
- `ts_argmax(x, n)` = x.rolling(n).apply(np.argmax)
- `ts_argmin(x, n)` = x.rolling(n).apply(np.argmin)

**横截面算子**：
- `rank(x)` = x在当天的横截面排名
- `scale(x, a)` = a * x / sum(abs(x))

#### Step 3: 因子组合策略

**主要组合方式**：

```python
# 1. 线性组合
def combine_factors_linear(factors, weights):
    """
    factors: dict, 因子名称 -> 因子值
    weights: dict, 因子名称 -> 权重
    """
    combined = sum(factors[name] * weights[name] for name in factors)
    return combined

# 2. 非线性组合（机器学习）
def combine_factors_ml(features, returns):
    """
    features: DataFrame, 多个因子
    returns: Series, 未来收益率
    """
    from sklearn.ensemble import RandomForestRegressor
    model = RandomForestRegressor()
    model.fit(features, returns)
    predictions = model.predict(features)
    return predictions

# 3. 动态权重（衰减）
def combine_factors_decay(factors, half_life):
    """
    factors: dict, 因子名称 -> 因子值
    half_life: 半衰期（月）
    """
    weights = {name: np.exp(-np.log(2) * age / half_life) for name, age in zip(factors, range(len(factors)))}
    total_weight = sum(weights.values())
    weights = {name: weight / total_weight for name, weight in weights.items()}
    combined = sum(factors[name] * weights[name] for name in factors)
    return combined
```

### 4.4 具体因子详解

#### 因子1: Alpha_001（趋势跟踪）

**计算逻辑**：
```python
Alpha_001 = ts_corr(log(volume), (close - open) / open, 6)
```

**详细解释**：
1. `log(volume)`：对成交量取自然对数，降低极端值影响
2. `(close - open) / open`：开盘价到收盘价的涨跌幅
3. `ts_corr(..., 6)`：计算6个交易日的滚动相关系数

**经济意义**：
- 正相关系数：放量上涨或缩量下跌（趋同）
- 负相关系数：放量下跌或缩量上涨（背离）

**回测效果**：
- IC值：0.032（1月）
- 年化收益率：9.2%
- 夏普比率：1.91
- 最大回撤：-3.8%
- 因子衰退：每年IC衰减约5%

#### 因子2: Alpha_002（成交量动量）

**计算逻辑**：
```python
Alpha_002 = ts_delta(rank(log(volume)), 2) * rank((close - open) / open)
```

**详细解释**：
1. `log(volume)`：成交量对数处理
2. `rank(log(volume))`：横截面排名
3. `ts_delta(..., 2)`：2日差分（环比变化）
4. `rank((close - open) / open)`：开盘收盘收益率排名
5. 相乘：成交量变化 × 价格变化

**经济意义**：
- 捕捉成交量与价格的协同效应
- 放量上涨类因子

**回测效果**：
- IC值：0.041（1月）
- 年化收益率：11.3%
- 夏普比率：2.32
- 最大回撤：-4.5%

#### 因子3: Alpha_003（价格波动率）

**计算逻辑**：
```python
Alpha_003 = ts_stddev(close, 10) / ts_mean(close, 10)
```

**详细解释**：
1. `ts_stddev(close, 10)`：10日收盘价标准差
2. `ts_mean(close, 10)`：10日收盘价均值
3. 比值：变异系数（CV）

**经济意义**：
- 衡量价格相对波动程度
- 低波动率股票倾向于未来表现更好（低波动异象）

**回测效果**：
- IC值：-0.028（负向因子）
- 年化收益率：7.8%
- 夏普比率：1.95
- 最大回撤：-3.1%

```python
def data_cleaning(df):
    """
    数据清洗函数
    """
    # 1. 剔除停牌数据
    df = df[df['volume'] > 0]
    
    # 2. 处理一字涨跌停（无法交易）
    df['is_limit'] = (df['high'] == df['low']) | (df['high'] == df['close']) | (df['low'] == df['close'])
    df.loc[df['is_limit'], ['close', 'open', 'high', 'low']] = np.nan
    
    # 3. 处理复权
    df['close'] = df['close'] * df['adj_factor']
    df['open'] = df['open'] * df['adj_factor']
    df['high'] = df['high'] * df['adj_factor']
    df['low'] = df['low'] * df['adj_factor']
    
    # 4. 异常值处理
    df = df[df['close'] > 0]
    df = df[df['volume'] > 0]
    
    return df
```

**教训**：

2. **样本内IC > 0.05的因子，样本外通常衰减50%**
3. **复杂因子比简单因子衰减更快**

**用户案例2：因子拥挤**（用户：因子老司机）

**问题**：
> "我的低波动因子2022年前效果很好，2023年后突然失效，回撤远超预期"

**分析**：
- 因子拥挤度检测（factor overcrowding）
- 2023年大量资金涌入低波因子
- 因子收益被摊薄，回撤增大

**解决思路**：
```python
# 因子拥挤度检测
def detect_factor_crowding(factor_returns, window=252):
    """
    检测因子拥挤度

    指标：
    1. 因子波动率（拥挤时上升）
    2. 因子换手率（拥挤时上升）
    3. 多头空头资金占比
    """
    # 1. 滚动波动率
    vol = factor_returns.rolling(window).std()

    # 2. 拥挤信号（波动率>2倍历史均值）
    crowding_signal = vol > 2 * vol.rolling(500).mean()

    # 3. 最近3个月拥挤天数
    crowding_days = crowding_signal.rolling(60).sum()

    return {
        'volatility_ratio': vol / vol.rolling(500).mean(),
        'crowding_signal': crowding_signal,
        'crowding_days_3m': crowding_days
    }

# 拥挤度>30天，考虑降低该因子权重
```

```python
def quality_screening(df):
    """
    质量筛选（规避雷区）
    """
    # 1. 剔除ST股
    df = df[~df['is_ST']]

    # 2. 剔除上市不满3年
    df = df[df['ipo_days'] > 365*3]

    # 3. 剔除总市值小于20亿（流动性风险）
    df = df[df['market_cap'] > 2e9]

    # 4. 剔除异常财务指标
    df = df[df['revenue_growth'] < 1000]  # 营收增长<10倍
    df = df[df['roe'] < 100]  # ROE<100%

    # 5. 剔除净利润连续下滑
    df = df[df['profit_growth'].rolling(3).mean() > -50]

    # 6. 现金流检验
    df = df[df['operating_cf'] > 0]  # 经营现金流为正

    return df
```
